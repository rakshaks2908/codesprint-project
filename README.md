# CodeSprint – Coding Practice & Quiz Platform

## About the Project

CodeSprint is a web-based platform built to help students manage their coding practice and test their understanding through quizzes in one place.

While working on coding regularly, I noticed that most platforms either focus only on solving problems or only on quizzes, but not both together. Because of this, tracking progress and consistency becomes difficult.

This project tries to solve that by combining coding practice tracking, quiz management, and performance analysis into a single system.

---

## Objective

The main goal of this project is to:

* Help students practice coding in a structured way
* Conduct timed quizzes for self-assessment
* Track performance and progress over time
* Provide a simple role-based system for managing content and users

---

## User Roles

### Student

* Register and login
* Upload solved coding problems
* Attempt quizzes within time limits
* View scores and quiz history
* Track personal progress

### Admin 1 (Quiz Manager)

* Create and manage quizzes
* Add, edit, delete questions
* Set time limits and rules

### Admin 2 (Performance Manager)

* View student performance
* Monitor quiz attempts
* Generate leaderboard
* Analyze overall progress

---

## How to Run the Project

1. Install XAMPP or any local server
2. Copy the project folder into htdocs
3. Start Apache and MySQL
4. Import the database into phpMyAdmin
5. Open browser and go to:
   http://localhost/your-folder-name

## Problem Statement

This project is based on the idea that students need a single platform to manage coding practice and quizzes together.

Most existing systems do not provide:

* Combined practice and assessment
* Proper tracking of progress
* Structured quiz management

CodeSprint aims to solve this by integrating all these features into one system.
